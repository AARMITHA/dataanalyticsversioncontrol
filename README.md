# Data Analytics Project Versioning

This repository demonstrates version control in a data analytics workflow using Git and GitHub.

### Files Included
- **Untitled11_(4).ipynb** – Jupyter notebook for analytics workflow  
- **data_preprocessing.py** – Handles cleaning and preprocessing tasks  
- **data_analysis.py** – Performs simple statistical analysis  
- **README.md** – Documentation  

### Overview
The project showcases how version control ensures reproducibility, collaboration, and organized tracking of progress in data analytics projects.
